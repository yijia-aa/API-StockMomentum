API to Search ResourceID

tablebuilder.singstat.gov.sg/api/table/resourceid

This calls the API using HTTPS GET.

Usage and SDK Samples

JavaScript

$(document).ready(function() {
  $.ajax({
      type: "GET",
      dataType: "json",
      url: "https://tablebuilder.singstat.gov.sg/api/table/resourceid?keyword=retail&searchOption=all",
      cache: false,
      success: function(data) {
        console.log(data);
      }
  });
});

Parameters

Query parameters

NameDescription
keyword*
*required
String

Keyword to be searched.
searchOption*
*required
Option include “all”, “title” or “variable”.

To search keyword within “All”, “Title” or “Variable.

Responses

Status: 200 - get resourceID

{
  Data: {
    generatedBy:      string
    dateGenerated:    string
    total:            integer
    records: [
      {
        id:           string
        tableType:    string
        title:        string
      }
    ]
  },
  DataCount:          integer
  StatusCode:         integer
  Message:            string
}

This calls the API using HTTPS GET.

API to Retrieve Table Data

tablebuilder.singstat.gov.sg/api/table/tabledata/{resourceId}

This calls the API using HTTPS GET.

Usage and SDK Samples

JavaScript

var resourceId      = 'M250141';
var seriesNoValue    = 13.1;
var offsetValue      = 1;
var limitValue       = 5;
var sortByValue      = 'value asc'
var timeFilterValue  = '2013';
var searchValue      = 'land';                                                
$(document).ready(function() {
  $.ajax({
      type: "GET",
      dataType: "json",
      url: "https://tablebuilder.singstat.gov.sg/api/table/tabledata/"+resourceId+"?seriesNoORrowNo="+seriesNoValue+"&offset="+offsetValue+"&limit="+limitValue+"&sortBy="+sortByValue+"&timeFilter="+timeFilterValue+"&search="+searchValue,
      cache: false,
      success: function(data) {
        console.log(data);
      }
  });
});

Parameters

Path parameters

NameDescription
resourceId*
*required
String

ID of the resource to be searched.

Query parameters

NameDescription
seriesNoORrowNo
String

seriesNo for Time Series Table and rowNo for Cross Sectional Table.
offset
Number

Specify the first n number of records to be excluded in the returned result.
limit
Number

Specify the number (maximum = 2000) of records to be included in the returned results for Time Series Table or number of rows to be included in the returned results for Cross Sectional Table.
sortBy
Option include key asc, key desc, value asc, value desc, seriesno asc, seriesno desc, rowno asc, rowno desc, rowtext asc, rowtext desc

Sort the returned records in descending or ascending order by the following fields: time, level, value, rowText, seriesNo, rowNo.

Not applicable for Cross Sectional tables
timeFilter
String

Return records of specific time points based on the type of selected table (monthly, half-yearly, quarterly, annual). There should be no spacing between commas. Example: For the quarterly table, timeFilter=2017 4Q,2018 1Q For the annual table, timeFilter=2017,2018 For the monthly table, timeFilter=2018 Mar For the half yearly table, timeFilter=2018 H1.

Not applicable for Cross Sectional tables
between
String

Specify a range within which the data values are to be filtered to give the returned result. The start and end points of the range are to be included. There should be no spacing between the values and commas.
search
String

Return records that contain the search string.

Responses

Status: 200 - get tabledata

Timeseries

{
  Data: {
    id:                 string
    title:              string
    footnote:           string
    frequency:          string
    datasource:         string
    generatedBy:        string
    dataLastUpdated:    string
    dateGenerated:      string
    offset:             string
    limit:              string
    sortBy:             string
    timeFilter:         string
    between:            string
    search:             string
    row: [
      {
        seriesNo:       string
        rowText:        string
        uom:            string
        footnote:       string
        columns:        [
          {
            key:         string
            value:       string
          }
        ]
      },
    ]
  },
  DataCount:            integer
  StatusCode:           integer
  Message:              string
}
